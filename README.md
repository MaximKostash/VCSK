**VCSK (Version control system Kostash)** - It is an easy to use system that will help you manage your project.

---
## Installation
To compile:
```
make
```
To install dependencies:
```
make depen
```
To install:
```
sudo make install
```
To uninstall:
```
sudo make uninstall
```
To clean temp files
```
make clean
```
